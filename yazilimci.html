Web Yazılımı Nedir?

Web yazılımı, tarayıcılar aracılığıyla görüntülenebilen dinamik ya da statik sayfalara verilen adlardır. Statik sayfalar artık pek kullanılmamakla beraber günümüzde daha çok dinamik web yazılımları kullanılmaktadır. Web yazılımları pek çok dil temel alınarak geliştirilebilmekte olup amaca göre farklı dizaynlara da sahip olabilmektedirler.

Web yazılımı ikiye ayrılır:

  1. Front-End
  2. Back-End

Front-End

Frontend'in Türkçe karşılığı “Önyüz”dür. Yapılma aşamasındaki bir web sitesinin ön yüzünü (client-side) HTML, CSS ve JavaScript gibi teknolojileri kullanarak web sitesinin görsel tarafını oluşturan kişilere ise front-end developer ( Ön yüz geliştirici ) denir

Back-End

Bir web projesinin son kullanıcının görmediği çekirdek yazılım kısmına Back-End ve bu mimariyi kodlayan yazılımcıya ise Back-End Developer, yani yazılım geliştirici denir.
